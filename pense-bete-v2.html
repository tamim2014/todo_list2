La 1ere colonne est trop mince!


				footer > div {
					flex: 1 1 250px !important; 
				}

Je veux aligner à gauche le contenu des div enfants
( avec une marge de 1em)


			    /* On cible seulement enfants  direct(>) */
				footer  > div {
					text-align:left;
					padding:1em;
				}

Comment ajouter un titre "h2" en haut du footer

1) Isoler h2
2) Mettre le reste dans un sous-contenair(.footer-row)
3) Migrer le flexbox: header → div.footer-row 

			/* On avait ce contenair, mais ça marche pas */
				footer{
					/*
					display:flex; 
					align-items:stretch;
					justify-content: center;
					*/
				}
			

			/* Alors on deplace le flexbox dans un sous-contenair*/
				footer .footer-row {
					display: flex;
					align-items:stretch; /* 📌 */
					justify-content: center;
				}

Comment ajouter un titre "h2" en haut du header?

1) Isoler h2
2) Mettre le reste dans un sous-contenair(.form-row)
3) Migrer le flexbox: header → div.form-row 

			/* On avait ce contenair, mais ça marche pas */
				header{
					/*
					display:flex; 
					align-items:center;  📌 
					justify-content: center;
					*/
				}
			

			/* Alors on deplace le flexbox dans un sous-contenair*/
				header .form-row {
					display: flex;
					justify-content: center;
				}

Pourquoi ces 3 elts ont la mm hauteur( alors qu'on a seulement défini Ajouter à 44px)?


		  /*  
			* Flexbox aligne les hauteurs automatiquement.
			*
			* Un conteneur flexbox applique par défaut:
			* ✔️ align-items: stretch;
			*
			*  Ce qui veut dire :
			*  "Tous les enfants prennent la même hauteur 
			*   que le plus grand."
			*	
			* Attention!
			* 👉 stretch(par défaut) ne s’applique que si 
			* 👉 les enfants n’ont PAS de hauteur définie
			*  			
			*/

Pourquoi le contenu du span ("Ajouter") n'est pas centré verticalement comme l'input et le select?


		  /*  
			* #task et #category sont des 
			* éléments de formulaire natifs (input, select)
			*
			* → leur rendu interne est géré par le navigateur
			* → le texte est centré verticalement dans la “zone de contrôle”, 
			*   pas juste dans la boîte CSS
			*				
			*/

Comment le centrer


				.addBtn { 
					display: flex;
					align-items: center;  /* ⭐📌 centrer verticalement */
					justify-content: center; /*centrer horizontalement */ 
				}

Pourquoi Flexbox marche aussi bien mm sur un p'tit btn(comme le btn "Ajouter")?

⭐ 📌Parce que Flexbox ne change pas la nature de l’élément.
⭐ 📌Il change juste comment son contenu interne est aligné.

Tu peux garder ça comme un mantra de dev front.
👉 Flexbox n’est pas un “layout global”,
👉 c’est un outil d’alignement, utilisable partout.
